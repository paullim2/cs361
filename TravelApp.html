This app can help plan your next trip! It provides details of your destination zipcode like population, distance, and temperature. If you're curious about the data, the following APIs were used to populate the data: ZipCode API (population), Google Maps Distance Matrix (distance), and OpenWeatherMap API (weather).

Show Instructions

Simply submit your current and destination zip codes in the Enter Zip Codes box and the details will present themselves in the Destination Details box. Accessibility options can be chosen in the bottom. Distance is shown in Miles by default and Temperature is shown in Fahrenheit by default. But you can choose the appropriate radio buttons to switch to Kilometer or Celcius.

Enter Zip Codes

Click submit to see the population, distance, and temperature of the destination zip code.
